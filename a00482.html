Detailed Description

Calibration index which contains the following components:

  • Linear equations for Inverter voltage / current
  • Power loss quadratic equations for up to 4 coils
  • Power loss FOD thresholds
  • Quality Factor scaling equations for up to 4 coils
  • Q-min quadratic equation
  • Open-air tank measurements
  • PID parameters for power regulation
  • Temperature thresholds for up to 4 sensors
