Detailed Description

ADC SFR mapping / APIs that scale with peripheral instances defined in qi_project_peripheral_defines.h.

If COIL_VOLT_DDM_PH or COIL_CURR_DDM_PH is defined, then digital filters for these channels will be enabled as signal is sampled over quarter quadrant and 4th sample is used for peak measurement for DDM.

void Drv_ADC_Init ( void  )

Initialize ADC module for Qi stack.

Parameters
void
Returns
void

Initializes the ADC module if not currently enabled. QI_Init_ADC() is used to configure/enable ADC triggers/interrupts for Qi stack.

Note
If ADC already enabled from application or another module, it is possible for ADC to be misconfigured for proper Qi stack operation. For example, wrong clock source/rate, core sampling time, etc.
