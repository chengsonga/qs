Detailed Description

Different FSK options and configurations. PWM interrupt service routine is used for calling FSK functions / counting PWM periods / etc. and this interrupt rate affects the modulation period (256/512 cycles). PWM interrupt is triggered every other cycle to reduce MIPS consumption and keep accurate FSK timings.
