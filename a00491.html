Detailed Description

Different linear equations for feedback channel.

Define feedback thresholds for Vbat, Vin, Vcoil, Icoil, etc. for given HW. y = mx+b with offset typically 0. These are default values to be loaded when Qi stack is initialized. Values may be overwritten with calibrated data.

General conversion:
Vadc_cnts (ADCBUFx) = Vc x Rdiv x Gadc (4095/3.3)
Vc = Vadc_cnts x 1/Rdiv x 1/Gadc
Vslope = 1/Rdiv x (4095/3.3) x Scaling units (1000) x max shift (i.e. 2^15)
