Detailed Description

Different quadratic equations for each coil and Q-min quadratic eqn.

Each coil in the application should have an independent Q-curve which is typically best fit with a quadratic equation. Const terms have been scaled by 10x to use provided APIs for quadratic where input/output is scaled by 10x.
