Detailed Description

SCCP modules configured for input capture mode with the DDM SCCPs configured to interrupt every edge and with a resolution of 1/4th the system clock (40ns). This is to ensure there isn't a chance of overflow as demodulated data timing nearing the max available count size 65535 (~650us).

For resonant frequency SCCP, a higher resolution is required and the data is only captured every rising edge as tied to comparator module with hysteresis on one edge.
