Detailed Description

PWM SFR mapping / APIs that scale with peripheral instances defined in qi_project_peripheral_defines.h. Two PWM generators operating in complementary mode are required to drive the full bridge inverter. One PWM (PHASE_SHIFT_PWM_GEN) is given a 180deg phase shift w.r.t MAIN_PWM_GEN. Default ping state frequency, duty cycle, and dead time are all set in qi_project_parameter_defines.h. MAIN_PWM_GEN uses TRIGC to sync start of cycle with the phase shifted generator. Other trigger sources are used for ADC sample and convert and are set inside drv_pwm.c.

Note
Inverter PWM generators must be within same group of 4. Either PWM1-PWM4 or PWM5-PWM8 group can be used for the two half-bridges. Configuration would need to be updated to use PCI sync features if not within same group of four.
